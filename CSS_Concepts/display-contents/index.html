The display Property

display: contents:

When the display of an element (call it 'a') is specified as contents, then all child elements of the element 'a' (element 'b' is one such element) will become children of the element that's one level up from 'a' in the DOM; so, here 'b' will become a child of the parent of element 'a' (in this case, the parent of element 'a' is the document itself). In effect, the styles applied to element 'a' will be disregarded.
