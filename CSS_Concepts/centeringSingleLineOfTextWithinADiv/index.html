When we have a div that holds a single line of text, for example a div containing 'Hello world!', a simple way to center text- both vertically and horizontally - in the div is to set the following properties in the div:

  1. give it width and height
  2. set text-align:center
  3. set the line-height to be the height of the div
  4. set vertical-align:middle
